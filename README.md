# AI Chat App

## Setup

### Backend

```bash
cd backend
npm install
```

Create `backend/.env` (see the provided email for values):

```env
MONGO_URI=
PORT=8080
GEMINI_API_KEY=
GEMINI_MODEL=gemini-2.0-flash
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

```bash
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
